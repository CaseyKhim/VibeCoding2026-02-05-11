
# Toto Number Generator

## Overview

A simple web application to generate random numbers for the Toto lottery.

## Features

*   **UI**: A clean and simple interface with a title, a display for the generated numbers, and a button to trigger the generation.
*   **Number Generation**: Generates a set of random numbers within a specified range.
*   **Styling**: Basic CSS for a pleasant user experience.

## Current Plan

*   Create the user interface in `index.html`.
*   Implement the number generation logic in `main.js`.
*   Style the application in `style.css`.
